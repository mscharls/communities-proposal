Included in the below Scope are all Project Management services for any required additional development team members. Task deadline management for internal Adept staff members is included for any additinal required inputs such as design assets, decisions, contract execution etc.


1. Research


The outputs from this phase will be:

  • A full list of user stories with priority rankings based on importance, effort and cost
  • A full understanding of the existing security and sharing models set up in Adept's production Org
  • Identification of Sandbox environment for development

    • 2. Design


    The outputs from this phase will be:

  • Full Solution Design including wireframes and any required customer journey mapping
  • Scope and analysis of Third Party Applications required to extend functionality to desired level
  • Breakdown of additional licensing costs for Third Party Applications
  • Internal user access defined
  • Development strategy, timelines, defined sprints and any required team members

    • 3. Develop

    4. Test

    5. Train

    6. Deploy


    The outputs from these phases will be:

  • Continued development and resolution of user stories through Agile sprints
  • Implementation and configuration of Third Party Applications
  • Testing on all functionality, security and sharing
  • Browser and device testing
  • System configuration documentation
  • Internal user training sessions on delivered functionality


    • Not Included

  • Configuration of Service Cloud features such as case management and live agent (Knowledge is included)
  • Creation of internal training manuals
  • Changes to scope not agreed to in writing
  • Custom Lightning component development not agreed to in writing
